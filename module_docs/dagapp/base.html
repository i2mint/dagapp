dagapp.base

The base components for making apps from DAGs

dagapp.base.get_pages_specs(dags, page_factory, configs)[source]

Return a mapping of page names to page callback objects.

The configs argument is an iterable of per-dag config dicts. If a config provides page_name or page_title that string will be used as the page name (and as the page title passed to the page factory). Otherwise a name is autogenerated from the dag via dag_to_page_name.
